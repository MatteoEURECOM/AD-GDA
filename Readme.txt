\f0\fs24 \cf0 This folder contains the code to reproduce the results presented in \'93Communication-Efficient Distributionally Robust Decentralized Learning\'94.\
\
Every sub-folder corresponds to one of the three experimental setups and it contains the implementations of AD-GDA, DRFA and DR-DSGD.\
\
To run in sequence all experimental runs of AD-GDA, run \'93script.py\'94.\
\
To run DRFA and DR-DSGD, run \'93network.py\'94.\
\
To run the experiments on COOS7 is necessary to first download the dataset from {\field{\*\fldinst{HYPERLINK "https://zenodo.org/record/3355599#.YzrxGy8Rr5g"}}{\fldrslt https://zenodo.org/record/3355599#.YzrxGy8Rr5g}}.\
